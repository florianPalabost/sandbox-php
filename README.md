# Sandbox PHP 8 - MySQL - nginx - PhpMyAdmin


## Requirements
- Docker, Docker-compose
- Make


## Install 
````
make install
# play in /html folder
````

## Stack
- PHP 8, Composer
- Nginx
- Mysql | Postgres
- PhpMyAdmin | PgAdmin

## Useful commands
The different commands are available in the ```/MakeFile``` file.
